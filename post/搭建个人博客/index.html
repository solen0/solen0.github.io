搭建个人博客

 2021-02-28 1360 words life blog hugo 

缘起

在学习途中,看的东西实在太杂,准备做些整理,想来若是写在纸上,不久就积灰了,况且我始终没把字练好,随即作罢

某日,忽的注意到别人的个人博客,甚是精美,于是惦记上了

此后通过囫囵的查找后,决定用Hugo搭建一个简洁的个人博客,并在闲暇时记录下我的旅途,顺便提升下我的写作水平[笑]

博客框架-Hugo

搭建个人博客,首先便是安装Hugo环境

  • 下载Hugo

下载完文件后,将其解压到合适位置

  • 配置环境变量

“此电脑"右键->属性->右侧的"高级系统设置”->环境变量->系统变量->“Path"中增加"hugo.exe"所在路径

在命令行中输入(win + R 输入cmd回车)

hugo version

显示版本号表示安装成功

用Hugo生成个人博客

“myblog"可任意设置

hugo new site myblog
cd myblog

寻找个人博客主题

主题模板

  • Fuji

  • hugo-theme-console

  • Clean-White

最终选择"Fuji"作为此博客主题

git clone https://github.com.cnpmjs.org/amzrk2/hugo-theme-fuji.git themes/fuji
cd themes
xcopy fuji\exampleSite\ ..\ /e /s
cd ..
hugo serve -t  fuji --buildDrafts

由于clone速度太慢,把"github.com"换成了"github.com.cnpmjs.org”(国内镜像)

其中”–buildDrafts",如果文章格式与主题默认文章格式一样可以不加,默认格式在"archetypes"中可修改,主题中可能是在"archives"或其它地方

此时打开 http://localhost:1313/ 可以看到博客模板已经布置好了

部署博客到GitHub

GitHub上建新仓库,名字前面要与用户名相同

博客文件夹下,开始部署

hugo --theme=fuji --baseUrl="https://你的用户名.github.io/" --buildDrafts    
cd public
git init                   
git add .
git commit -m "first commit"         
git remote add origin https://github.com/你的用户名/你的用户名.github.io.git
git push -u origin master

最后一步可能有网络问题(超时,连接重置),可以在hosts中加上GitHub IP地址(经常变,网上查)

之后就可以通过"你的用户名.github.io"访问博客啦

部署博客到gitee

在gitee上创建仓库

博客文件夹下,开始部署

hugo --theme=fuji --baseUrl="https://你的用户名.gitee.io/" --buildDrafts    
cd public
git init                   
git add .
git commit -m "first commit"
git remote add origin https://gitee.com/你的用户名/你的用户名.git
git push -u origin master

配置Gitee Pages

启动

之后就可以通过"https://你的用户名.gitee.io/“访问博客啦

写博客(以GitHub为例)

命令行中,在博客目录下(myblog),创建新文章

hugo new post\test.md

vscode 打开(C:\Users\用户名\博客名\content\post\test.md), 此时头部标签已经有了,往下就可以自由发挥了

  • 文章更新
hugo --theme=fuji --baseUrl="https://你的用户名.github.io/" --buildDrafts    
cd public
git init
git add -A
git commit -m "add posts"
git push

  1. “git push -f origin master” 本地强行覆盖远端,慎用

  2. “git push -u origin master” 将本地的master分支推送到origin主机，同时指定origin为默认主机

  3. “git push” 在 2 后用表示推送默认分支到默认主机

end
